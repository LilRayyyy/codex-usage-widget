# Changelog

## 0.1.0 - 2026-07-01

- Added the first open-source preview of Codex Usage Widget for macOS.
- Added a SwiftUI dashboard with overall and weekly remaining usage.
- Added desktop widgets for small, medium, and large sizes.
- Added current-month daily heatmap cells from day 1 through the end of month.
- Added app settings for refresh interval, menu bar icon, test data, Codex path, and Launch at Login.
- Added diagnostics for refresh status, App Group access, cache path, and Codex executable path.
- Added clearer widget empty states for missing cache, App Group setup, unsupported auth, and sign-in issues.
- Added 7-day usage trend and date-aware heatmap tooltips.
- Added app icon assets and local packaging guidance.
